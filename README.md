# art-description-generator

a web app to generate product descriptions in the art industry wit SEO optimised tags.
The user input the following information:
artist name
Artwork name
painting technique
year of production
size of painting
info about the painter (bio)
Language
link to image

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/art-description-generator.git
cd art-description-generator
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
